# Rootinize Premium AI Automation Agency

A modern, premium landing page for an AI automation agency, inspired by high-end SaaS/AI design aesthetics.

## Design Inspiration

This redesign extracts the visual essence from premium AI/SaaS templates, focusing on:

- **Dark Premium Aesthetic**: Deep dark backgrounds with refined contrast
- **Purple/Violet Accent Energy**: Gradient accents for visual impact
- **Glassy/Translucent Panels**: Modern glass morphism effects
- **Clean Spacious Composition**: Ample whitespace and elegant typography
- **Modern AI/SaaS Feel**: Conversion-focused design language
- **Subtle Animations**: Smooth transitions and hover effects

## Key Design Features

### Color Palette
- Primary Background: `#0a0a0f` (Deep Space Black)
- Secondary Background: `#11111f` (Dark Navy)
- Accent Gradient: `#8b5cf6` → `#6d28d9` (Purple/Violet)
- Text Colors: Refined contrast with `#f8fafc`, `#cbd5e1`, `#94a3b8`

### Typography
- Headings: **Space Grotesk** (Modern, geometric)
- Body: **Inter** (Clean, readable)
- Hierarchy: Clear visual hierarchy with proper scale

### UI Components
- **Glass Morphism Cards**: Translucent backgrounds with blur effects
- **Gradient Borders**: Subtle purple gradients on hover states
- **Animated Stats**: Counter animations for social proof
- **Interactive Elements**: Smooth hover and focus states
- **Responsive Design**: Fully responsive across all devices

### Animations & Effects
- **Fade-in Animations**: Elements appear on scroll
- **Float Animation**: Hero visual subtle movement
- **Hover Effects**: Cards lift and glow on hover
- **Loading States**: Interactive button feedback
- **Parallax Scrolling**: Subtle background movement

## Customization

### Colors
Edit the CSS variables in `style.css` at the `:root` selector:

```css
:root {
    --bg-primary: #0a0a0f;
    --accent-primary: #8b5cf6;
    /* ... other variables */
}
```

### Typography
Update the Google Fonts link in `index.html`:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800&family=Space+Grotesk:wght@400;500;600;700;800&display=swap" rel="stylesheet">
```

### Content
Modify the HTML structure in `index.html` to match your business needs.
